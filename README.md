# Interactive 3D Space Station

A real-time interactive 3D space station built with JavaScript, Three.js, and Vite. 
This project demonstrates 3D transformations, perspective projection, object animation, and camera navigation.

## 🚀 Tech Stack
* **Language:** JavaScript (ES6+)
* **3D Library:** [Three.js](https://threejs.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Version Control:** Git & GitHub

## 🛠️ Features (Assessment 2)
- [ ] Central space station core with docking modules and solar panels.
- [ ] Hierarchical rotation of the main station.
- [ ] Orbiting spacecraft with pause/resume controls.
- [ ] Free-roaming perspective camera with internal/external view toggles.

## 💻 Local Development
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local Vite development server.
